Strings (cadena de texto)

C√≥mo Python representa texto, trabaja con caracteres y permite manipular cadenas de forma flexible.

2.1 Creaci√≥n y longitud

Los strings se definen usando comillas y permiten representar texto. La funci√≥n len() se utiliza para conocer cu√°ntos caracteres contiene una cadena. 



var_string = "Mi cadena de texto"

print(len(var_second_string))  # Largo de la segunda cadena
üí°Buena pr√°ctica
Us√° nombres descriptivos como edad_usuario o total_ventas.

2.2 Formateo de strings


name, surname, age = "Juan", "Ignacio", 28

# Concatenaci√≥n con comas (funciona pero es feo) ‚Äî NO USAR
print("Mi nombre es ", name, surname, " y tengo ", age, " a√±os.")

# Interpolaci√≥n cl√°sica con .format() ‚Äî NO RECOMENDADO
print("Me llamo {} {} y tengo {} a√±os.".format(name, surname, age))

# Formato estilo C ‚Äî NO RECOMENDADO
print("Yo soy %s %s y tengo %d a√±os." % (name, surname, age))

# f-string ‚Äî FORMA MODERNA Y RECOMENDADA
print(f"Holis, yo soy {name} {surname} y tengo {age} a√±os.")
üí°Buena pr√°ctica
Us√° f para formatear texto: son m√°s legibles, modernos y expresivos.

2.3 Desempaquetado de caracteres

2.4 Slicing (rebanado)


print(lenguaje[1:3])     # 'yt'  ‚Üí desde √≠ndice 1 hasta 3 (sin incluir)

Listas

C√≥mo Python agrupa datos, mantiene colecciones ordenadas y permite acceder y modificar sus elementos.

3.1 Creaci√≥n y tipos


# Las listas son colecciones ordenadas y mutables.
# Se pueden crear con list() o con corchetes [], pero lo m√°s com√∫n es [].

mi_lista = list()   # Crea una lista vac√≠a (poco usado)
mi_otra_lista = []  # Crea una lista vac√≠a (forma recomendada)

# Sobrescribimos mi_lista con valores num√©ricos
mi_lista = [25, 33, 45, 18, 24, 26, 28, 17]

# Sobrescribimos mi_otra_lista con datos de distintos tipos
mi_otra_lista = [28, 1.79, 'Juan', 'Ignacio']

3.2 Informaci√≥n b√°sica


# --- Informaci√≥n sobre mi_lista ---
print(type(mi_lista))   # <class 'list'> ‚Üí es una lista
print(mi_lista)         # Muestra la lista completa
print(len(mi_lista))    # Cantidad de elementos en la lista

# --- Informaci√≥n sobre mi_otra_lista ---
print(mi_otra_lista)        # Muestra la lista
print(len(mi_otra_lista))   # Cantidad de elementos
print(type(mi_otra_lista))  # <class 'list'>

3.3 Acceso por √≠ndice


print(mi_otra_lista[0])   # Primer elemento ‚Üí 28
print(mi_otra_lista[1])   # Segundo elemento ‚Üí 1.79

# √çndices negativos: comienzan desde el final
print(mi_otra_lista[-1])  # √öltimo elemento ‚Üí 'Ignacio'
print(mi_otra_lista[-3])  # Tercero desde el final ‚Üí 1.79
print(mi_otra_lista[-4])  # Equivale al √≠ndice 0 ‚Üí 28

3.4 M√©todos de listas


# Lista de ejemplo
lista_num = [1, 2, 3, 4]

# append() ‚Üí Agrega un elemento al final
lista_num.append(5)
print(lista_num)   # [1, 2, 3, 4, 5]

# insert() ‚Üí Inserta un elemento en una posici√≥n espec√≠fica
lista_num.insert(1, 99)
print(lista_num)   # [1, 99, 2, 3, 4, 5]

# extend() ‚Üí Agrega varios elementos juntos (otra lista)
lista_num.extend([6, 7, 8])
print(lista_num)   # [1, 99, 2, 3, 4, 5, 6, 7, 8]

# pop() ‚Üí Elimina un elemento por √≠ndice
lista_num.pop()
print(lista_num)   # [1, 99, 2, 3, 4, 5, 6, 7]

lista_num.pop(1)
print(lista_num)   # [1, 2, 3, 4, 5, 6, 7]

# remove() ‚Üí Elimina la primera aparici√≥n de un valor
lista_num.remove(3)
print(lista_num)   # [1, 2, 4, 5, 6, 7]

# clear() ‚Üí Vac√≠a completamente la lista
# lista_num.clear()
# print(lista_num)  # []

# index() ‚Üí Devuelve el √≠ndice donde aparece un valor
lista_str = ["a", "b", "c", "b"]
print(lista_str.index("b"))   # 1

# count() ‚Üí Cuenta cu√°ntas veces aparece un valor
print(lista_str.count("b"))   # 2

# sort() ‚Üí Ordena la lista (modifica la original)
numeros = [4, 1, 7, 2]
numeros.sort()
print(numeros)   # [1, 2, 4, 7]

# reverse() ‚Üí Invierte el orden de la lista
numeros.reverse()
print(numeros)   # [7, 4, 2, 1]

3.5 Funciones √∫tiles con listas


# len() ‚Üí Devuelve la cantidad de elementos
len([10, 20, 30])              # 3

# sum() ‚Üí Suma todos los elementos num√©ricos
sum([10, 20, 30])              # 60

# max() ‚Üí Devuelve el valor m√°ximo
max([4, 9, 2, 7])              # 9

# min() ‚Üí Devuelve el valor m√≠nimo

# sorted() ‚Üí Devuelve una nueva lista ordenada (no modifica la original)

# list() ‚Üí Convierte un iterable en lista
list("hola")                   # ['h', 'o', 'l', 'a']

# type() ‚Üí Muestra el tipo del objeto
type([1, 2, 3])                # 

# any() ‚Üí True si al menos un elemento es True

# all() ‚Üí True si todos los elementos son True

# range() ‚Üí Genera una secuencia de n√∫meros (muy usado con listas)

üí°Buena pr√°ctica
Para crear listas transformadas, us√° list-comprehensions: son m√°s legibles y pythonicas que un for + append.

Tuplas (tuple)

Colecciones ordenadas e inmutables utilizadas para agrupar datos que no deben modificarse.

4.1 Concepto e inmutabilidad

Las tuplas son estructuras de datos **inmutables**. Son como las listas, pero **NO** permiten cambiar sus valores. 


mi_tupla = (10, 20, 30, 40)
print(mi_tupla)          # (10, 20, 30, 40)
print(type(mi_tupla))    #  <class 'tuple'>

# Acceso por √≠ndice
print(mi_tupla[0])       # 10
print(mi_tupla[2])       # 30

# Largo de la tupla
print(len(mi_tupla))     # 4

# Inmutabilidad
# mi_tupla[1] = 99  # ‚ùå ERROR: las tuplas no se pueden modificar
# Tampoco se pueden agregar ni borrar elementos

4.2 Tupla de un solo elemento



# Para crear una tupla con un solo elemento

Sets

Un set (conjunto) es una colecci√≥n de datos con las siguientes caracter√≠sticas:

  • No mantiene un orden definido
  • No utiliza √≠ndices
  • No permite elementos repetidos
  • Es modificable (se pueden agregar o quitar valores)

5.1 Concepto y creaci√≥n



# Set con elementos
mi_set = {"Juan", 20, True}
print("Set inicial:", mi_set)

# Set vac√≠o (¬°Ojo! {} crea un diccionario)
set_vacio = set()
print("Set vac√≠o:", set_vacio)

# mi_set[0]  # ‚ùå Esto da error porque no hay orden

5.2 Agregar y eliminar elementos



# AGREGAR ELEMENTOS
mi_set.add("Python")
print("Despu√©s de add('Python'):", mi_set)
# Si el elemento existe, no se duplica.

# ELIMINAR ELEMENTOS

# remove() elimina, pero da error si no est√°
mi_set.remove("Python")
print("Despu√©s de remove('Python'):", mi_set)

# discard() elimina, pero NO da error si no existe
mi_set.discard("Python")
print("Despu√©s de discard('Python') (no pasa nada):", mi_set)

# pop() elimina y devuelve un elemento aleatorio
elemento_eliminado = mi_set.pop()
print("Elemento eliminado con pop():", elemento_eliminado)
print("Set luego de pop():", mi_set)

# clear() vac√≠a el set por completo
mi_set.clear()
print("Set luego de clear():", mi_set)

# Intersecci√≥n

# Diferencia (elementos de set1 que no est√°n en set2)
diferencia = set1 - set2

# Diferencia sim√©trica (lo que no se repite entre ambos)
dif_simetrica = set1 ^ set2

Diccionarios

Los diccionarios funcionan con estructura clave: valor.

  • Las claves deben ser inmutables ( str, int, float, tuple)
  • Los valores pueden ser cualquier cosa: listas, funciones, diccionarios, etc.
